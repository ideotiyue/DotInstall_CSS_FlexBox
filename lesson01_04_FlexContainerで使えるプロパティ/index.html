フレックスボックス

Flexboxではflex container、flex itemにプロパティを設定することでレイアウトを組む。
プロパティごとに作用する軸が異なるのでどの方向に作用しているか意識してレイアウト組みをすること。

フレックスボックスのプロパティ

Flex Containerで使えるプロパティ

flex-direction

主軸の方向を決めるためのプロパティ。
初期値は row （行）。

flex-direction: row;

主軸は左から右。

flex item 1
flex item 2
flex item 3

flex-direction: row-reverse;

基本の形から主軸だけが反転

flex item 1
flex item 2
flex item 3

flex-direction: column;

基本の形から主軸と交差軸が逆

flex item 1
flex item 2
flex item 3

flex-direction: row-reverse;

主軸の向きが反対

flex item 1
flex item 2
flex item 3

justify-content

主軸に沿って要素をどう揃えるかを指定するためのプロパティ。
初期値は flex-start （行）。

justify-content: flex-start;

主軸の始点、つまり左揃え

flex item 3

justify-content: flex-end;

主軸の終点なので右揃え

justify-content: center;

主軸に沿って中央揃え

justify-content: space-between;

align-items

交差軸に沿ってどう要素を揃えるかの指定。
交差軸方向に余裕がある場合にのみ有効なプロパティ。
flexbox で align となっていたら、交差軸に対しての操作になる。

align-items: flex-start;

交差軸の始点なので上揃え

flex item 2
flex item 3

align-items: flex-end;

交差軸の終点なので下揃え

flex item 1
flex item 2
flex item 3

align-items: center;

交差軸に沿って中央揃え

flex item 1
flex item 2
flex item 3

flex-wrap

並べたい要素の幅がコンテナの幅を超える場合に使えるプロパティ。
初期値は nowrap。

flex-wrap: nowrap;

それぞれの幅が縮んで収まる。

flex item 1
flex item 2
flex item 3

flex-wrap: wrap;

はみ出した分を折り返す。

flex item 1
flex item 2
flex item 3

flex-wrap: wrap-reverse;

交差軸の向きが反転し、交差軸が始まる位置から主軸に沿って並べられていって、溢れたら更にコンテナが作られ、また主軸の方向に並べられる。
